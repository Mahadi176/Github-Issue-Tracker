1 What is the difference between var, let, and const?
Ans :

 Var is mainly function scoped it can be reassigned and redeclared .
Let is block scoped it reassigned but not redeclared.
const is also block scoped but cant be reassigned .

3️ What is the difference between map(), filter(), and forEach()?
Ans:

Map() is used to transform each elements and to return a new array.
Filter() is used select elements based on a condition and it returns a new array .
ForEach() is used to loop inside an array  to perform action but it does not return a new array.
